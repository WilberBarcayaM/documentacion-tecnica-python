Introduccion

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Fue creado en 1991 por Guido van Rossum y es ampliamente utilizado en la programación de scripts, desarrollo web, inteligencia artificial y ciencia de datos.

Python es un lenguaje muy popular debido a su sintaxis clara y fácil de entender, su amplia gama de bibliotecas y módulos disponibles y su capacidad para trabajar con diferentes sistemas operativos.

¿Para que es utiliza Python?

Python se utiliza para una gran variedad de aplicaciones, desde desarrollo web hasta análisis de datos, inteligencia artificial y aprendizaje automático. También es muy popular en la enseñanza de la programación debido a su sintaxis clara y legible.

Instalacion y Configuracion

Para empezar a programar en Python, primero debes instalar el intérprete de Python en tu ordenador. Puedes descargar la última versión de Python desde el sitio web oficial: https://www.python.org/downloads/

Una vez que hayas descargado e instalado Python, puedes ejecutar el intérprete de Python desde la línea de comandos. Para hacerlo, abre una ventana de terminal o consola y escribe 'python'. Esto te llevará al intérprete de Python, donde puedes escribir código Python y ejecutarlo.

Variables y tipos de datos

Una variable es un nombre que se utiliza para hacer referencia a un valor en la memoria de la computadora. En Python, no es necesario declarar una variable antes de utilizarla, simplemente se le asigna un valor.

Para asignar un valor a una variable en Python, puedes utilizar el signo de igual (=). Por ejemplo:

# Ejemplo de declaración de variables
x = 5
y = "Hola, mundo!"

Python admite varios tipos de datos, cada uno de los cuales puede ser útil en diferentes situaciones. A continuación, te explicaré los tipos de datos más comunes en Python:

  • Números

    En Python, existen tres tipos de números:

    1. Enteros (int): representan números enteros, como 1, 2, 3, etc.
    2. Números de punto flotante (float): representan números con decimales, como 1.2, 3.14, etc.
    3. Números complejos (complex): representan números complejos, como 3 + 4j, donde 3 es la parte real y 4j es la parte imaginaria.
    a = 10 # entero
    b = 3.14 # punto flotante
    c = 3 + 4j # complejo

  • Booleanos

    El tipo de dato 'bool' representa dos valores: 'True' (verdadero) y 'False' (falso). Estos valores son útiles en las expresiones condicionales y las operaciones lógicas.

    x = True
    y = False

  • Cadena de caracteres

    El tipo de dato 'str' representa una cadena de caracteres. Una cadena es una secuencia de caracteres entre comillas simples ('...') o comillas dobles ("...").

    saludo = 'Hola'
    nombre = "Juan"

  • Listas

    El tipo de dato 'list' representa una lista de elementos. Los elementos pueden ser de cualquier tipo de datos. Las listas son mutables, lo que significa que se pueden agregar, eliminar o modificar elementos después de su creación.

    mi_lista = [1, 2, 3, 'a', 'b', 'c']

  • Tuplas

    El tipo de dato 'tuple' representa una secuencia de elementos. Los elementos pueden ser de cualquier tipo de datos. A diferencia de las listas, las tuplas son inmutables, lo que significa que no se pueden agregar, eliminar o modificar elementos después de su creación.

    mi_tupla = (1, 2, 3, 'a', 'b', 'c')

  • Diccionarios

    El tipo de dato 'dict' representa un conjunto de pares clave-valor. Cada clave debe ser única en el diccionario, y los valores pueden ser de cualquier tipo de datos.

    mi_diccionario = {'nombre': 'Juan', 'edad': 25, 'ciudad': 'Madrid'}

  • Conjuntos

    El tipo de dato 'set' representa un conjunto de elementos únicos. Los elementos pueden ser de cualquier tipo de datos.

    mi_conjunto = {1, 2, 3, 'a', 'b', 'c'}

En Python, hay varios tipos de datos que se utilizan para representar diferentes tipos de información. Los tipos de datos más comunes son los números, booleanos, cadenas, listas, tuplas, diccionarios y conjuntos. Es importante comprender los diferentes tipos de datos y cómo se pueden utilizar en diferentes situaciones para escribir código eficiente y efectivo.

Operadores y expresiones

En Python, los operadores se utilizan para realizar operaciones sobre valores y variables. Las expresiones son combinaciones de valores, variables y operadores que producen un resultado.

A continuación, te presento los principales operadores y expresiones en Python:

  • Operadores aritméticos

    • '+' : suma dos valores
    • '-' : resta dos valores
    • '*' : multiplica dos valores
    • '/' : divide dos valores
    • '//' : devuelve la división entera de dos valores
    • '%' : devuelve el resto de la división de dos valores
    • '**' : eleva un valor a una potencia
    a = 10
    b = 5
    suma = a + b # 15
    resta = a - b # 5
    multiplicacion = a * b # 50
    division = a / b # 2.0
    division_entera = a // b # 2
    resto = a % b # 0
    potencia = a ** b # 100000

  • Operadores de comparación

    • '== ': devuelve 'True' si dos valores son iguales, 'False' en caso contrario
    • '!=' : devuelve 'True' si dos valores son diferentes, 'False' en caso contrario
    • '>' : devuelve 'True' si el primer valor es mayor que el segundo, 'False' en caso contrario
    • '<' : devuelve 'True' si el primer valor es menor que el segundo, 'False' en caso contrario
    • '>=' : devuelve 'True' si el primer valor es mayor o igual que el segundo, 'False' en caso contrario
    • <'=' : devuelve 'True' si el primer valor es menor o igual que el segundo, 'False' en caso contrario
    a = 10
    b = 5
    igual = a == b # False
    diferente = a != b # True
    mayor = a > b # True
    menor = a < b # False
    mayor_igual = a >= b # True
    menor_igual = a <= b # False

  • Operadores lógicos

    • 'and' : devuelve 'True' si ambas expresiones son verdaderas, 'False' en caso contrario
    • 'or' : devuelve 'True' si alguna de las expresiones es verdadera, 'False' en caso contrario
    • 'not' : devuelve el valor opuesto de la expresión
    a = 10
    b = 5
    c = 3
    y = a > b and b > c # True
    z = a < b or b < c # False
    w = not (a == b) # True

  • Expresiones condicionales

    Las expresiones condicionales se utilizan para tomar decisiones en función del resultado de una expresión. En Python, las expresiones condicionales se escriben utilizando el operador ternario 'if-else':

    valor = 10
    resultado = 'mayor' if valor > 5 else 'menor o igual'

Los operadores y expresiones son fundamentales en Python para realizar cálculos, comparaciones y tomar decisiones. Los operadores aritméticos se utilizan para realizar operaciones matemáticas, los operadores de comparación se utilizan para comparar valores y los operadores lógicos se utilizan para realizar operaciones lógicas

Estructuras de control

En Python, existen varias estructuras de control que nos permiten controlar el flujo de ejecución de un programa. Estas estructuras nos permiten tomar decisiones y repetir acciones en función de ciertas condiciones.

A continuación, te presento las principales estructuras de control en Python:

Estructura condicional if

La estructura condicional 'if' se utiliza para ejecutar una acción en caso de que se cumpla una condición. La sintaxis básica es la siguiente:

if condicion:
 # acciones a realizar si la condición es verdadera a = 10
if a > 5:
 print("a es mayor que 5")

También es posible utilizar la estructura if-else para ejecutar una acción en caso de que la condición no se cumpla:

if condicion:
 # acciones a realizar si la condición es verdadera
else:
 # acciones a realizar si la condición es falsa

Ejemplo:

a = 3
if a > 5:
 print("a es mayor que 5")
else:
 print("a es menor o igual que 5")

También es posible utilizar la estructura 'if-elif-else' para evaluar varias condiciones en cascada:

if condicion1:
 # acciones a realizar si la condición 1 es verdadera
elif condicion2:
 # acciones a realizar si la condición 2 es verdadera
else:
 # acciones a realizar si ninguna condición es verdadera

Ejemplo:

a = 10
if a > 15:
 print("a es mayor que 15")
elif a > 5:
 print("a es mayor que 5 pero menor o igual que 15")
else:
 print("a es menor o igual que 5")

Estructuras de repetición while y for

Las estructuras de repetición nos permiten ejecutar un bloque de código varias veces en función de ciertas condiciones.

La estructura 'while' se utiliza para repetir un bloque de código mientras se cumpla una condición:

while condicion:
 # acciones a realizar mientras la condición sea verdadera

Ejemplo:

a = 0
while a < 5:
 print(a)
 a += 1

La estructura 'for' se utiliza para iterar sobre una secuencia de elementos:

for elemento in secuencia:
 # acciones a realizar para cada elemento de la secuencia

Ejemplo:

numeros = [1, 2, 3, 4, 5]
for numero in numeros:
 print(numero)

Control de flujo break y continue

El control de flujo 'break' se utiliza para salir de una estructura de repetición en el momento en que se cumple una cierta condición:

while condicion:
 if condicion_salida:
  break
 # acciones a realizar mientras la condición sea verdadera

Ejemplo:

a = 0
while True:
 if a >= 5:
  break
 print(a)
 a += 1

El control de flujo continue se utiliza para saltar a la siguiente iteración en una estructura de repetición en el momento en que se cumple una cierta condición:

for elemento in secuencia:
 if condicion_salto:
  continue
 # acciones a realizar para cada elemento de la secuencia
Funciones

Las funciones en Python son bloques de código que pueden ser llamados en cualquier lugar del programa para realizar una tarea específica. Las funciones pueden tomar argumentos de entrada y pueden devolver un resultado.

Aquí hay un ejemplo de una función en Python que toma dos argumentos y devuelve su suma:

def sumar(a, b):
 resultado = a + b
 return resultado

En este ejemplo, la función 'sumar' toma dos argumentos 'a' y 'b'. Dentro de la función, se realiza la operación de suma y el resultado se almacena en la variable 'resultado'. Luego, la función devuelve el valor de 'resultado'.

Para llamar a esta función, podemos hacer lo siguiente:

resultado_suma = sumar(2, 3)
print(resultado_suma) # Salida: 5

Aquí, llamamos a la función 'sumar' con los argumentos '2' y '3'. La función devuelve '5', que se almacena en la variable 'resultado_suma' y luego se imprime en la pantalla.

Además de los argumentos, las funciones en Python también pueden tener valores de parámetros predeterminados y argumentos de longitud variable. Aquí hay un ejemplo de una función con argumentos de longitud variable:

def imprimir_nombres(*nombres):
 for nombre in nombres:
  print(nombre)

En este ejemplo, la función 'imprimir_nombres' toma un número variable de argumentos 'nombres'. Dentro de la función, se utiliza un bucle 'for' para iterar a través de cada uno de los argumentos y se imprime cada uno de ellos en la pantalla.

Para llamar a esta función, podemos hacer lo siguiente:

imprimir_nombres('Juan', 'María', 'Carlos')

Aquí, llamamos a la función 'imprimir_nombres' con tres argumentos: 'Juan', 'María' y 'Carlos'. La función itera a través de los argumentos y los imprime en la pantalla.

Programacion orientada a objetos

La programación orientada a objetos (POO) es un paradigma de programación que se basa en la creación de objetos que tienen propiedades y métodos asociados. Python es un lenguaje orientado a objetos, lo que significa que se pueden crear clases y objetos en el programa.

Para crear una clase en Python, se utiliza la palabra clave 'class'. Aquí hay un ejemplo de una clase 'Persona' que tiene una propiedad 'nombre' y un método 'saludar':

class Persona:
 def __init__(self, nombre):
  elf.nombre = nombre
 def saludar(self):
  print(f'Hola, mi nombre es {self.nombre}')

En este ejemplo, la clase Persona tiene un constructor '__init__ ' que toma un argumento 'nombre'. La propiedad nombre se inicializa con el valor del argumento 'nombre'. La clase también tiene un método 'saludar' que imprime un mensaje de saludo con el valor de 'nombre'.

Para crear un objeto de la clase 'Persona', se utiliza el nombre de la clase seguido de paréntesis. Aquí hay un ejemplo de cómo crear un objeto 'persona1' de la clase 'Persona':

persona1 = Persona('Juan') persona1.saludar() # Salida: Hola, mi nombre es Juan

En este ejemplo, creamos un objeto 'persona1' de la clase 'Persona' con el nombre 'Juan'. Luego llamamos al método 'saludar' del objeto, que imprime el mensaje de saludo con el nombre 'Juan'.

Además de las propiedades y métodos, las clases en Python también pueden tener métodos especiales como '__str__' para representar un objeto como una cadena y '__len__' para obtener la longitud de un objeto. También se pueden crear subclases y heredar propiedades y métodos de una clase principal.

La programación orientada a objetos en Python puede ser un concepto complejo, pero es una herramienta poderosa para organizar y estructurar programas.

Herencia

La herencia es una característica importante de la programación orientada a objetos que permite crear nuevas clases basadas en una clase existente, con la capacidad de agregar o modificar sus propias propiedades y métodos.

En Python, para crear una subclase que hereda de una clase principal, simplemente se define una nueva clase y se especifica la clase principal entre paréntesis después del nombre de la nueva clase. Por ejemplo, si tenemos una clase 'Animal' que tiene una propiedad 'nombre' y un método 'hacer_sonido', podemos crear una subclase 'Perro' que hereda de la clase 'Animal' y agrega un nuevo método 'ladrar':

class Animal:
 def __init__(self, nombre):
  self.nombre = nombre
 def hacer_sonido(self):
  pass # Este método se define en las subclases
class Perro(Animal):
 def ladrar(self):
  print('¡Guau!')

En este ejemplo, la subclase 'Perro' hereda de la clase 'Animal' mediante la especificación de 'Animal' entre paréntesis después del nombre de la nueva clase. La subclase 'Perro' también tiene un nuevo método 'ladrar' que imprime el mensaje '¡Guau!'.

Podemos crear objetos de la clase Perro y llamar a sus métodos:

mi_perro = Perro('Fido')
print(mi_perro.nombre) # Salida: Fido
mi_perro.hacer_sonido() # Este método no hace nada en esta clase
mi_perro.ladrar() # Salida: ¡Guau!

En este ejemplo, creamos un objeto 'mi_perro' de la clase 'Perro' con el nombre 'Fido'. Podemos acceder a la propiedad 'nombre' del objeto, que es heredada de la clase 'Animal'. También llamamos al método 'hacer_sonido', que está definido en la clase principal 'Animal', pero no hace nada en la subclase 'Perro'. Finalmente, llamamos al método 'ladrar', que es específico de la subclase 'Perro'.

La herencia en Python es una herramienta muy útil para organizar y estructurar programas orientados a objetos. Permite reutilizar código y definir clases más específicas a partir de clases generales existentes.

Modulos y paquetes

En Python, un módulo es un archivo que contiene código de Python que se puede reutilizar en otros programas. Un paquete es una colección de módulos relacionados que se organizan en una estructura de directorios.

Los módulos y paquetes en Python se importan en otros programas usando la palabra clave 'import'. Por ejemplo, si tenemos un módulo llamado 'mimodulo.py' con una función llamada 'mi_funcion', podemos importar la función en otro programa de la siguiente manera:

import mimodulo

mimodulo.mi_funcion()

En este ejemplo, estamos importando el módulo mimodulo y llamando a su función mi_funcion.
También es posible importar sólo una función específica de un módulo, usando la sintaxis from modulo import funcion. Por ejemplo, si queremos importar sólo la función mi_funcion del módulo mimodulo, podemos hacerlo de la siguiente manera:

from mimodulo import mi_funcion

mi_funcion()

En este ejemplo, estamos importando la función 'mi_funcion' directamente del módulo 'mimodulo' y llamándola directamente sin tener que usar el nombre del módulo.

A continuación te explico un poco más sobre los módulos y paquetes más utilizados en Python:


Módulo math

El módulo 'math' proporciona funciones matemáticas básicas, como trigonométricas, exponenciales, logarítmicas, entre otras. A continuación, un ejemplo de cómo utilizar algunas de estas funciones:

import math

x = 2
y = 3

# Funciones trigonométricas
print(math.sin(x))
print(math.cos(x))
print(math.tan(x))
print(math.atan(x))

# Funciones exponenciales y logarítmicas
print(math.exp(x))
print(math.log(x))
print(math.log10(x))

# Otras funciones
print(math.sqrt(x))
print(math.ceil(x/y))
print(math.floor(x/y))

Módulo NumPy

NumPy es un paquete que proporciona herramientas para trabajar con arrays multidimensionales y matrices. Es muy utilizado en la computación científica y el análisis de datos. A continuación, un ejemplo de cómo utilizar NumPy para crear un array y realizar algunas operaciones:

import numpy as np

# Crear un array
x = np.array([1, 2, 3, 4, 5])

# Operaciones con arrays
print(x + 1)
print(x * 2)
print(np.sin(x))

# Operaciones entre arrays
y = np.array([5, 4, 3, 2, 1])
print(x + y)
print(x * y)
print(np.dot(x, y))

Módulo pandas

Pandas es un paquete que proporciona herramientas para el análisis de datos y la manipulación de datos en formato tabular. Es muy utilizado en la ciencia de datos y el aprendizaje automático. A continuación, un ejemplo de cómo utilizar pandas para cargar un archivo CSV y realizar algunas operaciones:

import pandas as pd

# Cargar un archivo CSV
data = pd.read_csv("datos.csv")

# Mostrar los primeros registros
print(data.head())

# Selección de columnas
x = data["columna1"]
y = data["columna2"]

# Operaciones entre columnas
z = x + y
print(z)

Los módulos y paquetes son una parte fundamental de Python, ya que proporcionan herramientas para trabajar con diferentes tipos de datos y realizar diferentes operaciones. El módulo 'math' proporciona funciones matemáticas básicas, NumPy proporciona herramientas para trabajar con arrays multidimensionales y matrices, y pandas proporciona herramientas para el análisis de datos y la manipulación de datos en formato tabular. Estos son solo algunos de los muchos módulos y paquetes disponibles en Python, y es importante explorar y utilizar diferentes herramientas según las necesidades de cada proyecto.
